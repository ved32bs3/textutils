# TextUtils & Form Builder

A responsive React web application designed for fast text manipulation and interactive writable form generation. Built with React and styled using Bootstrap, it features a customizable theme engine to modify the light-mode color scheme.

## Features

* **Writable Forms**: Dynamic input fields that users can fill out, edit, and reset.
* **Text Transformation**: Convert text to uppercase or lowercase instantly.
* **Whitespace Remover**: Clean up unnecessary spaces from your text.
* **Copy & Clear**: One-click actions to copy text to the clipboard or clear the workspace.
* **Live Metrics**: Real-time count of words, characters, and estimated reading time.
* **Theme Customizer**: Settings panel to change the application's light theme accent color.

## Tech Stack

* **Framework**: React (Functional Components & Hooks)
* **Styling**: Bootstrap 5
* **Icons**: Bootstrap Icons

## Installation

Follow these steps to run the project locally.

### Prerequisites

Ensure you have Node.js installed on your system.

### Steps

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```

2. Navigate to the project directory:
   ```bash
   cd textutils
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.
